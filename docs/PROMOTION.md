# 🎨 Domain Theme Router

> **一套内容，无限可能** —— 让你的 Halo 站点在不同域名下展现不同风采

---

## 🚀 你是否遇到过这些困扰？

- 有优质内容，但想在不同域名下展示不同风格？
- 运营多个网站，却要重复发布相同内容？
- 想做 A/B 测试，但搭建多个站点太麻烦？
- 企业有多个品牌，希望统一管理内容？

**Domain Theme Router 帮你一键解决！**

---

## ✨ 核心亮点

### 🎯 一石多鸟
一套内容，多个域名，不同主题。无需重复发布，无需多站点管理。

### ⚡ 极速体验
基于 WebFilter 实现，请求处理性能影响微乎其微。用户无感知，体验零延迟。

### 🔌 即插即用
无需修改代码，无需复杂配置。安装插件，配置域名，立即生效。

### 🛡️ 稳定可靠
智能主题验证，自动回退机制。主题不可用时，自动切换到默认主题，确保站点始终可用。

### 🎨 灵活配置
支持精确域名匹配，智能格式处理。带不带协议、大小写、端口号，统统帮你搞定。

```

```

不同地区，不同审美，统一内容管理。

## 💡 工作原理

```
┌─────────────────────────────────────────────────────┐
│                    用户访问域名                        │
└─────────────────────┬───────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────┐
│               Domain Theme Router                    │
│  ┌──────────────────────────────────────────────┐  │
│  │  1. 读取请求域名 (Host / X-Forwarded-Host)    │  │
│  │  2. 匹配域名绑定配置                          │  │
│  │  3. 应用对应主题                              │  │
│  └──────────────────────────────────────────────┘  │
└─────────────────────┬───────────────────────────────┘
                      │
                      ▼
┌─────────────────────────────────────────────────────┐
│              Halo 内容管理系统                        │
│  • 统一的文章、页面、菜单                             │
│  • 统一的附件、用户管理                               │
│  • 统一的后台管理界面                                 │
└─────────────────────────────────────────────────────┘
```

**核心思想**：内容与展示分离，域名决定风格。

## 🛠️ 快速开始

### 3 分钟完成配置

```bash
# 1. 下载插件
wget https://github.com/muyin-dev/halo-plugin-domain-theme-router/releases/latest/download/halo-plugin-domain-theme-router.jar

# 2. 安装插件（在 Halo 后台上传）
# 后台 → 插件 → 安装插件 → 上传 jar 文件

# 3. 配置域名绑定
# 后台 → 插件详情 → 设置 → 添加绑定

# 4. 配置 DNS/反向代理
# 确保域名正确解析并传递正确的 Host 头
```

### 配置示例

```yaml
# 域名绑定配置
bindings:
  - domain: "blog.example.com"
    themeName: "theme-simple"
    enabled: true
    remark: "简约风格博客"
    
  - domain: "news.example.com"  
    themeName: "theme-news"
    enabled: true
    remark: "新闻资讯风格"
    
  - domain: "docs.example.com"
    themeName: "theme-docs"
    enabled: false
    remark: "文档中心（暂未启用）"
```
